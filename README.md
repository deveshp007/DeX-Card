## Tech Stack/Features

- Tech Stacks:
  - Material Design
  - Kotlin
  - Splash-screen(using API)
  - Minimum SDK level 21

- Customizations:
  - Themes
  - Hide specific stat
  - Adding private commits
  - Hide icons
  - Hide ranks
  - ...many more.
